# Configuration

Once the services are running, you can access the Lilac frontend by navigating to `http://localhost:3000` in your web browser.

From here, you can begin to configure your data pipelines. For more detailed information on configuring the different components of Lilac, please refer to the following sections:

- **[Agent Configuration](/agent)**
- **[Backend Configuration](/backend)**
- **[Frontend Configuration](/frontend)**

Congratulations! You have successfully installed and configured Lilac. You are now ready to start building your data pipelines.
